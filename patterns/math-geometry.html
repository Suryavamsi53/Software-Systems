üß© 5. Canonical Software Systems Problem
Rotate Image (LeetCode 48)

Problem: You are given an n x n 2D matrix representing an image, rotate the image by 90 degrees (clockwise). You have to rotate the image in-place.

Input: matrix = [[1,2,3],[4,5,6],[7,8,9]]
Output: [[7,4,1],[8,5,2],[9,6,3]]

üî¥ 6. Brute Force Analysis
Extra Matrix

Create a new matrix and copy elements: new_matrix[j][n-1-i] = matrix[i][j].

Space Complexity: O(N¬≤) - Uses extra space.

Why Software Systems Rejects: The constraint usually specifies in-place modification.

üü¢ 7. Optimized Solution
func rotate(matrix [][]int) {
    n := len(matrix)
    
    // 1. Transpose
    for i := 0; i < n; i++ {
        for j := i; j < n; j++ {
            matrix[i][j], matrix[j][i] = matrix[j][i], matrix[i][j]
        }
    }
    
    // 2. Reverse Rows
    for i := 0; i < n; i++ {
        for j := 0; j < n/2; j++ {
            matrix[i][j], matrix[i][n-1-j] = matrix[i][n-1-j], matrix[i][j]
        }
    }
}

class Solution {
    public void rotate(int[][] matrix) {
        int n = matrix.length;
        
        // Transpose
        for (int i = 0; i < n; i++) {
            for (int j = i; j < n; j++) {
                int temp = matrix[i][j];
                matrix[i][j] = matrix[j][i];
                matrix[j][i] = temp;
            }
        }
        
        // Reverse Rows
        for (int i = 0; i < n; i++) {
            for (int j = 0; j < n/2; j++) {
                int temp = matrix[i][j];
                matrix[i][j] = matrix[i][n-1-j];
                matrix[i][n-1-j] = temp;
            }
        }
    }
}

class Solution:
    def rotate(self, matrix: List[List[int]]) -> None:
        n = len(matrix)
        # Transpose
        for i in range(n):
            for j in range(i, n):
                matrix[i][j], matrix[j][i] = matrix[j][i], matrix[i][j]
        
        # Reverse rows
        for i in range(n):
            matrix[i].reverse()

‚öôÔ∏è 9. Complexity Analysis
Time & Space

Time Complexity: O(N¬≤)

We visit each cell twice (once for transpose, once for reverse).

Space Complexity: O(1)

We modify the matrix in-place without extra storage.

üü£ 10. Software Systems Follow-up Questions
Variations

Q: How to rotate anti-clockwise?
A: Transpose first, then reverse columns (or reverse rows first, then transpose).

Q: What if the matrix is not square (MxN)?
A: You cannot rotate in-place easily. You need a new matrix of size NxM.
