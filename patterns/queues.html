Queues (FIFO)

First-In, First-Out Data Structure

ğŸ§  Core Concept
FIFO Property

A Queue is a linear data structure that follows the First-In, First-Out principle. Elements are added at the rear (enqueue) and removed from the front (dequeue).

Use Cases:

  • BFS: Breadth-First Search in graphs/trees.
  • Task Scheduling: Processing jobs in order.
  • Buffering: IO buffers, message queues.

type Queue struct {
    items []int
}

func (q *Queue) Enqueue(i int) {
    q.items = append(q.items, i)
}

func (q *Queue) Dequeue() int {
    if len(q.items) == 0 {
        return -1
    }
    item := q.items[0]
    q.items = q.items[1:]
    return item
}
