Sliding Window Pattern

Master contiguous subarray and substring problems

üîç How to Spot This Pattern: The 3-Prong Detection System
üìä INPUT SIGNAL
  • Array or String
  • Sequential data
  • Need contiguous part
  • Stream of data
‚ùì QUESTION SIGNAL
  • Longest/Shortest substring
  • Max sum subarray of size K
  • String anagrams
  • Contains permutation
‚ö° CONSTRAINT SIGNAL
  • O(n) time complexity
  • O(1) or O(k) space
  • Avoid re-calculating overlap
  • Single pass preferred
‚úÖ 1. What Software Systems EXPECTS
Interviewer's Checklist
  • Window Type: Can you identify if it's a Fixed Size or Dynamic Size window?
  • Optimization: Do you avoid re-calculating the sum/count for the overlapping part?
  • Data Structure: Do you use a Hash Map or Set correctly to track window state?
  • Complexity: Can you explain why the solution is O(n) even with nested loops (amortized)?
üß† 2. Core Concept Theory
Don't Re-calculate

The Sliding Window pattern is used to perform a required operation on a specific window size of a given array or string. Instead of re-calculating the entire window, we slide it by one element: Subtract the element leaving and Add the element entering.

Types:

  • Fixed Window: Window size k is constant. (e.g., Max sum of 3 elements).
  • Dynamic Window: Window grows/shrinks based on constraints. (e.g., Smallest subarray with sum >= S).
üéØ 3. Real-World Software Systems Scenario
Rate Limiting & Monitoring

Scenario: Software Systems API Gateway needs to limit users to 100 requests per minute. Or, CloudWatch needs to calculate the average CPU usage over the last 5 minutes.

Solution: A sliding window of timestamps allows us to count requests in the current time frame efficiently, dropping old requests as time moves forward.

üß© 5. Canonical Software Systems Problem
Longest Substring Without Repeating Characters (LeetCode 3)

Problem: Given a string s, find the length of the longest substring without repeating characters.

Input: s = "abcabcbb"
Output: 3 (The answer is "abc", with the length of 3.)

üî¥ 6. Brute Force Analysis
Check All Substrings

Generate all possible substrings and check if they have unique characters.

Time Complexity: O(N¬≥) - O(N¬≤) substrings * O(N) to check uniqueness.

Why Software Systems Rejects: Extremely inefficient for long strings.

üü¢ 7. Optimized Solution
func lengthOfLongestSubstring(s string) int {
    charIndex := make(map[byte]int)
    left := 0
    maxLength := 0

    for right := 0; right < len(s); right++ {
        char := s[right]
        
        // If char is in map and is inside the current window
        if idx, ok := charIndex[char]; ok && idx >= left {
            // Move left pointer to skip the duplicate
            left = idx + 1
        }
        
        // Update the last seen index of the character
        charIndex[char] = right
        
        // Calculate current window length
        currentLength := right - left + 1
        if currentLength > maxLength {
            maxLength = currentLength
        }
    }
    
    return maxLength
}

class Solution {
    public int lengthOfLongestSubstring(String s) {
        Map map = new HashMap<>();
        int left = 0;
        int maxLength = 0;
        
        for (int right = 0; right < s.length(); right++) {
            char c = s.charAt(right);
            
            if (map.containsKey(c)) {
                // Move left pointer to max of current left or duplicate's next index
                left = Math.max(left, map.get(c) + 1);
            }
            
            map.put(c, right);
            maxLength = Math.max(maxLength, right - left + 1);
        }
        
        return maxLength;
    }
}

class Solution:
    def lengthOfLongestSubstring(self, s: str) -> int:
        char_index = {}
        left = 0
        max_length = 0
        
        for right, char in enumerate(s):
            if char in char_index and char_index[char] >= left:
                left = char_index[char] + 1
            
            char_index[char] = right
            max_length = max(max_length, right - left + 1)
            
        return max_length

‚öôÔ∏è 9. Complexity Analysis
Time & Space

Time Complexity: O(N)

We traverse the string once. Each character is added and removed from the window at most once.

Space Complexity: O(min(N, M))

Where M is the size of the charset (e.g., 26 for lowercase English, 128 for ASCII). We store the last index of each character.

üü£ 10. Software Systems Follow-up Questions
Variations

Q: What if we want the longest substring with at most K distinct characters?
A: Use a map to count frequencies. When map size > K, shrink left until map size <= K.

Q: What if the input is a stream?
A: We can't keep the whole string. We might need to store only the current window in a buffer.
