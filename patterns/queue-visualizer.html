📚 Prerequisites
  • FIFO Principle: Queues strictly operate on a First-In, First-Out basis. The first element enqueued is the first to be dequeued.
  • Circular Buffer Math: When implemented with a static array, instead of shifting elements left during a dequeue (which takes O(N) time), we use Modulo Arithmetic: (index + 1) % capacity to efficiently wrap the Head and Tail pointers around the array in O(1) time.
  • Pointers: A Queue uses two pointers—a Head (front reading point) and a Tail (rear insertion point).
💡 Pro Interview Tips
  • BFS Integration: Queues are the fundamental data structure used to implement Breadth-First Search (BFS) for Trees and Graphs. (e.g., Level Order Traversal).
  • Language Implementations:
    - Java: Use Queue<E> q = new LinkedList<>();
    - Python: Use collections.deque for O(1) ops. Python lists are bad for queues because list.pop(0) is O(N).
  • Edge Cases: Like stacks, always check for Underflow (dequeue on empty) and Overflow (enqueue on full fixed buffer).
