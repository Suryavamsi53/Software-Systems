üì± Mobile Access Guide

How to study patterns on your phone or tablet over WiFi

üöÄ The Easy Way (Recommended)
Use the Helper Script

We've included scripts that automatically find your IP address and start the server.

  1. Open your terminal in the project folder.
  2. Mac/Linux: Run this command:
    sh start.sh
  3. Windows: Double-click start.bat or run:
    start.bat
  4. Look for the line that says "On your mobile".
  5. Type that exact URL into your phone's browser (Chrome/Safari).
üõ†Ô∏è Manual Method
1. Ensure Same Network

Your computer and mobile device must be connected to the same WiFi network.

2. Find Your Local IP
Mac

Run in terminal:
ipconfig getifaddr en0

Windows

Run in cmd:
ipconfig
(Look for IPv4 Address)

Linux

Run in terminal:
hostname -I

3. Connect

On your phone, visit: http://YOUR_IP_ADDRESS:8000

Example: http://192.168.1.45:8000

üåç Access from Anywhere (Different Network)

If you are not on the same WiFi, you can use a tunneling service to expose your local server to the internet securely.

Option 1: Using ngrok (Recommended)
  1. Download and install ngrok.
  2. Start your local server on port 8000 (as usual).
  3. In a new terminal window, run: ngrok http 8000
  4. Copy the https://....ngrok-free.app URL provided.
  5. Open that URL on any device, anywhere in the world.
Option 2: Static Hosting (Permanent)

Since this is a static website, you can host it for free on GitHub Pages, Netlify, or Vercel.

‚ö†Ô∏è Troubleshooting
If it doesn't load...
  • Firewall: Your computer's firewall might block port 8000. Try temporarily disabling it or allowing Python.
  • Network Isolation: Some "Guest" WiFi networks prevent devices from talking to each other. Use the main WiFi.
  • Wrong IP: IP addresses can change. Re-run serve.py to check.
