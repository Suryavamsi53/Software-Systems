üó∫Ô∏è DSA Roadmap

A structured 8-week path to crack Amazon SDE interviews

‚≠ê How to use this roadmap

This roadmap is designed for depth over breadth. Don't just memorize solutions. For each topic:

  1. Understand the core concept (Theory)
  2. Learn the pattern (Templates)
  3. Solve the canonical problem
  4. Practice 3-5 variations
Phase 1: The Foundation Weeks 1-2

Mastering the basics of arrays, strings, and complexity analysis. These are the building blocks for everything else.

Complexity Analysis Big O, Time vs Space, Logarithmic time
Two Pointers Pairs, Reversing, Partitioning
Sliding Window Subarrays, Substrings, Anagrams
Prefix Sum Range queries, Cumulative sums
Phase 2: Core Data Structures Weeks 3-4

Essential structures for organizing data. Hash Maps are the most important tool in your arsenal.

Hash Maps & Sets O(1) lookups, Counting, Deduplication
Linked Lists Traversal, Reversal, Fast & Slow Pointers
Stacks & Queues LIFO/FIFO, Monotonic Stack, BFS Queue
Phase 3: Search & Sorting Week 5

Efficiently finding and organizing data. Binary search is critical for optimization problems.

Binary Search Sorted arrays, Search space reduction
Sorting & Searching Merge Sort, Quick Sort, Custom Comparators
Intervals Merging, Overlaps, Scheduling
Phase 4: Non-Linear Structures Week 6

Hierarchical and connected data. Trees and Graphs appear in almost every Amazon onsite loop.

Trees & BST DFS, BFS, Recursion, Level-order
Graphs Adjacency List, Islands, Shortest Path
Heaps (Priority Queue) Top-K elements, Median, Scheduling
Tries Prefix matching, Autocomplete
Phase 5: Advanced Algorithms Week 7

Optimization and exhaustive search. These distinguish SDE-2 candidates.

Dynamic Programming Memoization, Tabulation, Knapsack
Backtracking Permutations, Subsets, Puzzles
Greedy Local optimization, Activity selection
Phase 6: Final Polish Week 8

Specialized topics and behavioral prep.

Mock Interviews Behavioral (LP), Coding, System Design
System Design Scalability, Load Balancing, Caching
Bit Manipulation XOR, Bit masking
