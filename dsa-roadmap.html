üó∫Ô∏è Comprehensive DSA Roadmap

A pattern-based curriculum covering everything from Arrays to Advanced Graphs.

üìò Teaching Philosophy

We follow a strict Pattern-Based Approach designed for FAANG interviews:

  • Intuition First: Visualizing the concept before writing code.
  • Pattern Recognition: Identifying the pattern from problem signals.
  • Visual Explanations: Using ASCII diagrams and step-by-step flows.
  • Interview Focus: Prioritizing clarity, communication, and trade-offs.

üéì How to Use This Roadmap

1. Learn the Pattern

Don't memorize code. Understand the visual intuition and when to use it.

2. Solve Canonical

Implement the classic problem (e.g., Two Sum) from scratch without looking.

3. Apply to Variations

Solve 3-5 similar problems to cement the pattern recognition.

4. Spaced Repetition

Revisit the pattern after 3 days, then 1 week to retain it.

The foundation of all DSA interviews. Master these first.

Key Intuition: Most array problems can be solved by iterating with pointers (Two Pointers, Sliding Window) or storing frequency (Hash Maps).

Opposite ends, Fast & Slow, Merging sorted arrays

Fixed size, Variable size, String permutations

Top Questions Range Sum Query, Subarray Sum Equals K, Product of Array Except Self
Intervals Medium
Overlapping intervals, Insert Interval, Meeting Rooms
Top Questions Merge Intervals, Insert Interval, Non-overlapping Intervals, Meeting Rooms II
Hashing / Hashmaps Easy
Frequency counting, Two Sum, Grouping Anagrams
Top Questions Two Sum, Group Anagrams, Longest Consecutive Sequence, Subarray Sum Equals K

Searching efficiently in sorted (and unsorted) spaces.

Key Intuition: Ask yourself: "Can I eliminate half the search space?" If yes, use Binary Search. Works on sorted arrays and "answer spaces".

Pointer manipulation mastery.

Key Trick: Use a Dummy Node to simplify edge cases (like deleting the head). Use Fast & Slow Pointers for cycles and middle finding.
